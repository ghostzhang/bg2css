CSS Sprites 样式生成工具 快捷键

菜单快捷键列表
名称 快捷键 说明
打开图片 Ctrl + O 打开要操作的图片。
更新图片 F5 更新当前图片文件。
读取定义文件 Ctrl + Shift + O 打开之前保存的定义文件。
保存当前定义 Ctrl + S 保存当前的定义到文件。
当前层信息 Ctrl + N 显示｜隐藏当前层的信息面板。
所有层信息 Ctrl + Shift + N 打当前层的信息窗口。
显示区定义 Ctrl + M 显示｜隐藏显示区的定义面板。
生成样式 Ctrl + P 显示｜隐藏生成样式的定义面板。
样式预览 Ctrl + Shift + P 显示｜隐藏样式预览面板。
图片操作快捷键列表
名称 快捷键 说明
图片缩放 Ctrl + 0
Ctrl + 1 		图片100%显示。
Ctrl + 2 图片200%显示。
Ctrl + 4 图片400%显示。
Ctrl + 6 图片600%显示。
当前层操作快捷键列表
名称 快捷键 说明
删除 Delete | (fn + delete) 删除当前层。
移动当前层 Ctrl + 向上方向键 向上移动当前层的位置。
Ctrl + 向下方向键 向下移动当前层的位置。
Ctrl + 向左方向键 向左移动当前层的位置。
Ctrl + 向右方向键 向右移动当前层的位置。
修改当前层 Ctrl + Shift + 向上方向键 减少当前层的高度。
Ctrl + Shift + 向下方向键 增加当前层的高度。
Ctrl + Shift + 向左方向键 减少当前层的宽度。
Ctrl + Shift + 向右方向键 增加当前层的宽度。
