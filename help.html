CSS Sprites 样式生成工具 快捷键

菜单快捷键列表
名称 快捷键(win) 快捷键(mac) 说明
打开图片 Ctrl + O command + O 打开要操作的图片。
更新图片 F5 无 更新当前图片文件。
读取定义文件 Ctrl + Shift + O command + Shift + O 打开之前保存的定义文件。
保存当前定义 Ctrl + S command + S 保存当前的定义到文件。
当前层信息 Ctrl + N command + N 显示｜隐藏当前层的信息面板。
所有层信息 Ctrl + Shift + N 无 打当前层的信息窗口。
显示区定义 Ctrl + M 无 显示｜隐藏显示区的定义面板。
生成样式 Ctrl + P command + P 显示｜隐藏生成样式的定义面板。
样式预览 Ctrl + Shift + P command + Shift + P 显示｜隐藏样式预览面板。

Ctrl + Shift + 向下方向键 cntrol + command + Shift + 向下方向键 增加当前层的高度
